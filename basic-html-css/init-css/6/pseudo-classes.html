Pseudo Classes

Hover

.text:hover { background-color: bisque; }

First Child

.list p:first-child { color: white; }

Nth-child

.siblings p:nth-child(2n + 1) { color: cornflowerblue; }

Link : Mean the a is not visited yet

.link a:link { color: blue; }

/* Visited: Link visited */

.link a:visited { color: green; }

Active: Link active is the link is clicked

.link a:active { color: crimson; }

Focus

.input:focus { background: yellow; color: blue; }
